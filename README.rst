yavimim
=======

yet another vim input method

**现在属于alpha阶段！！！**

按键绑定
--------

启/停用中文输入快捷键： ``<C-_>``

中文输入状态下按键绑定
^^^^^^^^^^^^^^^^^^^^^^^

+---------------+------------------+--------------+
| 键            | 插入模式         | 非插入模式   |
+---------------+------------------+--------------+
| ``<Enter>``   | 换行             | 换行         |
+---------------+------------------+--------------+
| ``<C-E>``     | 清除已输入字符   |              |
+---------------+------------------+--------------+
| ``<ESC>``     | 退出插入模式     | 退出当前输出 |
+---------------+------------------+--------------+
| ``0`` - ``9`` | 选字上档         |              |
+---------------+------------------+--------------+
| ``<Space>``   | 输出空格或者上档 |              |
+---------------+------------------+--------------+
| ``-``         | 上一页           |              |
+---------------+------------------+--------------+
| ``=``         | 下一页           |              |
+---------------+------------------+--------------+
| ``;``         | 英文模式         | 英文模式     |
+---------------+------------------+--------------+
